# Gapura UB Mobile

Aplikasi mobile portal akademik Universitas Brawijaya berbasis Flutter. Dikembangkan sebagai bagian dari penelitian skripsi untuk mengevaluasi dan memperbaiki desain antarmuka pengguna.

## Tentang Aplikasi

Gapura UB Mobile memberikan akses layanan dan informasi akademik bagi mahasiswa Universitas Brawijaya melalui perangkat mobile. Aplikasi ini dirancang dengan memperhatikan pengalaman pengguna (UX) dan aksesibilitas, termasuk dukungan mode gelap (dark mode).

## Fitur Utama

- **Beranda** — ringkasan jadwal kelas hari ini, menu utama, dan pengumuman kelas
- **Akademik** — akses layanan akademik seperti Kartu Hasil Studi (KHS) dan Jadwal Presensi
- **Presensi Kelas** — pencatatan kehadiran dengan status Hadir, Izin, Sakit, dan Terlewat
- **Kartu Hasil Studi** — rekap nilai dan informasi IP per semester
- **Kartu Rencana Studi** — informasi mata kuliah yang diambil
- **Informasi** — portal berita dan informasi seputar universitas
- **Profil** — informasi akun mahasiswa

## Teknologi

- **Framework:** Flutter (Dart)
- **Platform:** Android
- **Tema:** Light Mode & Dark Mode

## Catatan

Beberapa fitur dalam aplikasi ini berada di luar scope penelitian dan ditandai sebagai tidak tersedia pada versi ini.
